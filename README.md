### 1. 🤖 Predictive Engineering & Machine Learning
Detailed algorithmic implementations tracking specific industry objectives.

* **📈 Loan Prediction Profile**
  * **Objective:** Automate cross-verification checkpoints for risk mitigation in loan profiling.
  * **Methods & Algorithms:** Handled historical classification tracking, feature scaling via `MinMaxScaler`, and evaluation using ROC-AUC thresholds.
  * **Repository:** [`synent-task8-Loan-Prediction-hasyapatel`](https://github.com/Hasya1505/synent-task8-Loan-Prediction-hasyapatel)

* **⏱️ Time-Series Analysis**
  * **Objective:** Extrapolate trend lines and seasonality variations across sequenced datasets.
  * **Methods & Algorithms:** Applied ARIMA models and stationary tests (ADF Test) to validate temporal parameters.
  * **Repository:** [`synent-task-7--Time-Series-Analysis-hasyapatel`](https://github.com/Hasya1505/synent-task-7--Time-Series-Analysis-hasyapatel)

* **🏠 House Price Exploration**
  * **Objective:** Multi-variable feature regression evaluating physical asset variables on local real estate pricing matrix.
  * **Methods & Algorithms:** Multicollinearity reduction using VIF (Variance Inflation Factor) and Gradient Boosting Regressors.
  * **Repository:** [`house-price-exploration`](https://github.com/Hasya1505/house-price-exploration)

* **🧮 Mathematical Foundations of ML**
  * **Objective:** Complete mathematical foundation repository featuring custom implementation of gradient descent, distance functions, and loss parameters without standard library wrappers.
